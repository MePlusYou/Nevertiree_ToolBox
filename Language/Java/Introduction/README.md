# Introduction

## Basic Type

### Integer

byte/Byte 1

short/Short 2

int/Integer  4

long/Long 8

十进制为0x开头，八进制为0开头，long为L结尾。

### Float Point

float/Float 4

double/Double 8

float类型用F结尾

### boolean/Boolean

true 1

false 1

### String

char/Character 2

char类型的值用两个单引号括起来，String类型用两个双引号括起来（但是字符串类型不属于基本类型数据）

Java能够从小往大自动转换，如果出现了自大往小转换的情况，需要手动进行强制转换

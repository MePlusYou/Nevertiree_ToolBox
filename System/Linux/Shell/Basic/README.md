## 注释

以“#”开头的行就是注释，会被解释器忽略。sh里没有多行注释，只能每一行加一个#号。

如果在开发过程中，遇到大段的代码需要临时注释起来，过一会儿又取消注释，可以把这一段要注释的代码用一对花括号括起来，定义成一个函数，这块代码就不会执行，达到了和注释一样的效果。
